# ai201-project4-provenance-guard

Provenance Guard is a service that estimates whether a piece of creative writing
was likely written by a human or generated by AI, and returns a cautious,
human-readable attribution label.

## Running locally

```bash
source .venv/bin/activate
python app.py
```

The app serves on port 5000. On macOS, `localhost:5000` may be intercepted by the
AirPlay Receiver — use `http://127.0.0.1:5000` in your curl commands if requests
return empty responses.

## Milestone 4: Multi-Signal Confidence Scoring

Provenance Guard now uses two independent detection signals:

1. **Groq LLM Signal** (`groq_llm_signal`): evaluates the text holistically with a
   cautious prompt and returns an AI-likelihood score from 0.0 to 1.0.
2. **Stylometric Heuristic Signal** (`stylometric_signal`): calculates structural
   writing features — sentence length variance, type-token ratio, punctuation
   density, and repetition ratio — and maps them to an AI-likelihood score. This
   signal asks no model for judgment; it measures the text directly.

The two signals are combined into a single confidence score:

```text
combined_score = (0.65 * groq_score) + (0.35 * stylometric_score)
```

The combined score maps to three attribution categories:

| Score Range | Attribution   |
| ----------- | ------------- |
| 0.80 – 1.00 | likely_ai     |
| 0.30 – 0.79 | uncertain     |
| 0.00 – 0.29 | likely_human  |

The uncertainty band is intentionally wide because false positives are harmful on a
creative-writing platform. A score like 0.60 does not mean the content is
AI-generated — it means the system found mixed evidence and returns `uncertain`.

### Audit log

Every classification is appended to `audit_log.jsonl` (viewable at `GET /log`) with
both individual signal scores recorded:

- `groq_score`, `groq_label`, `groq_reasoning`
- `stylometric_score`, `stylometric_label`, `stylometric_features`
- `confidence` (combined score) and final `attribution`

### Testing

Run the stylometric signal in isolation:

```bash
python test_stylometry.py
```

Or exercise the full pipeline against the running server:

```bash
curl -s -X POST http://127.0.0.1:5000/submit \
  -H "Content-Type: application/json" \
  -d '{"text": "...", "creator_id": "test-user-1", "title": "My Test"}' \
  | python -m json.tool
```
